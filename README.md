# QuantitativeFinance

1. Derivatives Pricers Project

A modular Python application for pricing financial derivatives with a modern graphical interface.
The project implements analytical pricing models such as Black–Scholes to value different derivatives while following a clean, scalable architecture.

The system separates financial models, application logic, and user interfaces, making it easy to extend with additional products, pricing models, or market data sources. A REST API enables programmatic access to the pricing engine, while a modern web GUI allows users to interactively input parameters and compute option prices.
